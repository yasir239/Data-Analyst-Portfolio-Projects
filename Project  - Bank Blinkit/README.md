# Blinkit Grocery Data

## Project Summary
This project analyzes grocery delivery data to evaluate sales performance, customer ordering behavior, and operational KPIs. The goal is to identify opportunities to improve fulfilment efficiency and increase revenue per order.

## Key Analyses
- Sales performance over time (daily/weekly/monthly)
- Top-selling categories and SKU-level trends
- Customer order frequency and retention analysis
- Delivery time and fulfilment efficiency metrics
- Revenue by store/zone and channel

## Tools & Technologies
- Data cleaning & exploration: Excel, Python (pandas)
- Dashboarding & reporting: Power BI / Excel dashboards
- Visualization: matplotlib / seaborn (for reproducible analysis)

## Data
Primary dataset: `BlinkIT Grocery Data.csv` (order-level and product-level fields)

## How to Use
1. Open the dataset in Excel or load into Python for analysis.
2. Explore pivot tables or Power BI report for interactive dashboards.
3. Review the `Power BI` dashboard for KPIs and trends (if included).
